# System Documentation
## Fashion Store Static Website

---

# Introduction

Fashion Store adalah website statis yang digunakan sebagai media katalog produk fashion. Website menampilkan berbagai koleksi pakaian dengan desain modern, sederhana, dan responsif sehingga dapat diakses melalui berbagai perangkat.

---

# Objectives

- Menampilkan katalog produk fashion.
- Memperkenalkan brand kepada pelanggan.
- Memberikan informasi mengenai produk.
- Menyediakan media promosi online.
- Menampilkan informasi kontak toko.

---

- Pengunjung Website

# Functional Requirements

FR-01

Website menampilkan halaman Home.

FR-02

Website menampilkan daftar produk.

FR-03

Pengguna dapat mencari produk.

FR-04

Pengguna dapat memfilter produk berdasarkan kategori.

FR-05

Pengguna dapat melihat detail produk.

FR-06

Website menampilkan informasi perusahaan.

FR-07

Website menampilkan halaman kontak.

---

# Non Functional Requirements

Performance

- Loading halaman kurang dari 3 detik.

Availability

- Website dapat diakses 24 jam.

Compatibility

- Google Chrome
- Microsoft Edge
- Mozilla Firefox
- Safari

Responsive

- Desktop
- Laptop
- Tablet
- Smartphone

Usability

- Navigasi sederhana dan mudah dipahami.
- Tampilan konsisten pada setiap halaman.
